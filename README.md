
# Установка и использование

## Установить как cli-скрипт

```bash
git clone git@github.com:6umpukc/6umpukc.git ~/bin/6umpukc && cd ~/bin/6umpukc/bin && chmod +x bx && MSYS=winsymlinks:native ./bx install
```

## Список доступных команд

`bx help`

## Описание команды

`bx help [command]`

например:

`bx help install-php`
